# 🌤️ Weather Dashboard

A modern, responsive weather dashboard built with React, Vite, and Tailwind CSS. Fetches real-time weather data from OpenWeatherMap API.

## ✨ Features

- ✅ Real-time weather data
- ✅ 5-day forecast
- ✅ City search functionality
- ✅ Geolocation support
- ✅ Responsive mobile design
- ✅ Beautiful UI with Tailwind CSS
- ✅ Error handling & loading states

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
git clone https://github.com/Hope-land321/weather-dashboard.git
cd weather-dashboard

npm install
